Desenvolvi este simulador como parte de um desafio da DIO com o objetivo de aplicar, na prática, os conceitos de Excel em um contexto financeiro real.
O objetivo deste projeto foi criar uma planilha simples e funcional para simular investimentos em Fundos Imobiliários.
A ideia foi aplicar os conhecimentos de Excel para calcular automaticamente o total investido, o patrimônio acumulado e uma estimativa de dividendos mensais, facilitando a visualização do crescimento do investimento ao longo do tempo.
Durante o desenvolvimento deste projeto, consegui:
Aplicar juros compostos de forma prática
Utilizar funções financeiras do Excel com mais segurança
Estruturar uma planilha pensando em organização e clareza
Automatizar cálculos para apoiar tomada de decisão
Documentar um projeto técnico de forma profissional no GitHub
Esse desafio reforçou para mim como dados bem organizados e metas claras fazem diferença.
A experiência também contribuiu para desenvolver organização, raciocínio lógico e visão analítica, aplicando Excel como ferramenta de simulação financeira.
